# Interwor · Valoració de Ciberseguretat

App web per a l'avaluació de l'estat de ciberseguretat dels clients. Permet als usuaris respondre un qüestionari de 101 controls de seguretat agrupats en 9 categories, i genera informes visuals amb recomanacions de productes.

## Funcionalitats

- **Formulari interactiu** amb 101 controls i 9 categories
- **Bilingüe** català/castellà amb selector d'idioma
- **Botó d'info** amb explicació detallada de cada control
- **Gràfics** radar i barres de compliment per categoria
- **Recomanacions** automàtiques de productes basades en les mancances
- **Importar/Exportar Excel** compatible amb la plantilla original
- **Exportar a PDF** amb informe complet i branding Interwor

## Executar en local

```bash
npm install
npm run dev
```

## Desplegar a GitHub Pages

El projecte inclou un workflow de GitHub Actions que es desplega automàticament a cada push a `main`. Activa GitHub Pages a Settings > Pages > Source: GitHub Actions.

## Tecnologies

React 18 + Vite + SheetJS (xlsx)
